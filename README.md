# webnovel_downloader
Download webnovel chapters as an epub file for reading offline

## How to Run
- Modify the "NOVEL_URL" to point to the main page of the novel from novelfull.com/net
- Modify the "OUTPUT_FILENAME" to be the name you want to save with a .epub ending
- Run the main.py program

## Features
- Download each chapter as a json file
- Resume in case of crash/pause/stop
- Rerun to download missing files
- Automatically extract cover image from main page
- Combine everything to epub format
